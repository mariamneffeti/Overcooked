# Cultural Pulse v3

## 4-Layer Architecture

1. **Layer 1 — Signal Capture (`ingestion/`)**: Scrapers and early signal processing.
2. **Layer 2 — Analytics (`intelligence/`)**: Pattern modeling and context extraction.
3. **Layer 3 — Insights (`intelligence/workers/`)**: Summaries, recommendations, and strategy prompts.
4. **Layer 4 — Generative Outputs (`action_engine/`)**: Creative assets and ROI forecasting.

## Supporting Systems

- **Backend (`backend/`)**: API gateway, middleware, and data stores.
- **Frontend (`frontend/`)**: Monitoring dashboards and briefing pages.
- **Shared (`shared/`)**: Reusable constants and schemas.
